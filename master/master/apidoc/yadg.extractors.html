yadg.extractors package

yadg.extractors.extract(filetype, path)

The individual extractor functionality of yadg is called from here.

Extracts data from provided path, assuming it is the specified filetype. The data is either returned as a DataTree or a Dataset. In either case the returned objects have a to_netcdf() as well as a to_dict() method, which can be used to write the object into a file.

Parameters:

  • filetype (str) – Specifies the filetype. Has to be a filetype supported by the dataschema.

  • path (Path) – A pathlib.Path object pointing to the file to be extracted.

Return type:

Union[Dataset, DataTree]
